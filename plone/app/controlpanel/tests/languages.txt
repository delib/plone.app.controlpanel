Languages control panel
========================

First some initial setup code:

    >>> from Products.CMFCore.utils import getToolByName
    >>> portal_languages = getToolByName(self.portal, 'portal_languages')
    >>> self.loginAsManager()

Viewing the language control panel
------------------------------------

    >>> self.browser.open('http://nohost/plone/@@language-controlpanel')
    >>> self.browser.url.endswith('language-controlpanel')
    True

Click the cancel button:

    >>> self.browser.getControl(name="form.actions.cancel").click()
    >>> self.browser.url.endswith('plone_control_panel')
    True

There should be still no changes:

    >>> 'Changes canceled.' in self.browser.contents
    True

Make some changes
-----------------

    >>> self.browser.open('http://nohost/plone/@@language-controlpanel')
    >>> self.browser.url.endswith('language-controlpanel')
    True

    >>> self.browser.getControl(name='form.use_combined_language_codes').value = 'checked'

Click the save button:

    >>> self.browser.getControl(name="form.actions.save").click()
    >>> self.browser.url.endswith('language-controlpanel')
    True

We should be informed that something has changed:

    >>> 'Changes saved.' in self.browser.contents
    True

Make sure the changes have been applied correctly and combined language codes
are shown in the browser:

    >>> portal_languages.use_combined_language_codes
    True

    >>> 'pt-br' in self.browser.contents
    True
